**Tribe**
- id_tribe (PK)
- name (UNIQUE)
- founding_date
- max_members (CHECK > 0)

**Player**
- id_player (PK)
- username (UNIQUE)
- level (CHECK 1–100)
- join_date
- id_tribe (FK, nullable — player pode não ter tribo)

**Creature**
- id_creature (PK)
- nickname
- level (CHECK 1–150)
- sex (CHECK: 'M' / 'F')
- status (CHECK: 'wild' / 'tamed' / 'dead')
- health
- stamina
- id_specie (FK)
- id_player (FK, nullable — criatura pode ser selvagem)

**Specie**
- id_specie (PK)
- name (UNIQUE)
- diet (CHECK: 'carnivore' / 'herbivore' / 'omnivore' / 'piscivore')
- threat_level (CHECK 1–5)
- can_fly (boolean)
- can_swim (boolean)

**Map**
- id_map (PK)
- name (UNIQUE)
- biome (CHECK: 'jungle' / 'arctic' / 'desert' / 'ocean' / 'volcanic')
- difficulty (CHECK 1–5)

**Mutation**
- id_mutation (PK)
- attribute (CHECK: 'health' / 'stamina' / 'melee' / 'speed' / 'weight')
- description

**Item**
- id_item (PK)
- name (UNIQUE)
- type (CHECK: 'kibble' / 'narcotic' / 'food' / 'tool')
- rarity (CHECK: 'common' / 'uncommon' / 'rare' / 'legendary')

---

**Entidades associativas:**

**Domestication** (Player + Creature)
- id_domestication (PK)
- id_player (FK)
- id_creature (FK)
- tame_date
- method (CHECK: 'passive' / 'violent')
- effectiveness (CHECK 0–100)
- time_spent_min

**Creature_mutation** (Creature + Mutation)
- id_creature (FK)
- id_mutation (FK)
- generation (CHECK >= 1)
- value (o quanto aquele atributo foi incrementado)

**Spawn_rate** (Specie + Map)
- id_specie (FK)
- id_map (FK)
- frequency (CHECK: 'rare' / 'uncommon' / 'common')

**Inventory** (Player + Item)
- id_player (FK)
- id_item (FK)
- quantity (CHECK > 0)
